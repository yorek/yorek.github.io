# personal-homepage

Minimal static personal website for GitHub Pages.

## Edit content

- Main content: `index.html`
- Styles: `styles.css`

## Run locally

Open `index.html` directly in your browser, or use any static server.

## Deploy on GitHub Pages

1. Push to your repository default branch.
2. In GitHub: `Settings` -> `Pages`.
3. Set source to `Deploy from a branch`.
4. Select your default branch and `/ (root)` folder.

`CNAME` is already configured for `www.davidemauri.it`.
